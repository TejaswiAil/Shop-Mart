#  Amazon Clone (Laravel + Vue + Inertia)

This project is a **modern e-commerce platform** modeled after Amazon, built using **Laravel**, **Vue.js**, **Inertia.js**, and **Tailwind CSS**. 

- **Backend:** Laravel (PHP)
- **Frontend:** Vue.js + Inertia.js
- **Styling:** Tailwind CSS
- **Database:** MySQL 
- **Others:** Eloquent ORM
